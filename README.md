# ⏱️ Stopwatch Web Application

A simple and responsive Stopwatch application built using **HTML, CSS, and JavaScript**. This project allows users to start, stop, reset, and record lap times with an attractive user interface.

## 🚀 Features

* Start the stopwatch
* Stop/Pause the stopwatch
* Reset the stopwatch
* Record lap times
* Responsive and modern UI
* Smooth button hover effects
* Scrollable lap history section

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

* Lap time recording section
* Gradient background and modern card design

## ⚙️ How It Works

### Start Button

Starts the stopwatch and updates the timer every second.

### Stop Button

Pauses the stopwatch without resetting the current time.

### Reset Button

Stops the timer, resets the display to `00:00:00`, and clears all recorded laps.

### Lap Button

Stores the current stopwatch time in the lap history list.
